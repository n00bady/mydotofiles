# My dotfiles
These are my dot files, there are many like them but these are mine :)

## Hyprland config
Most of the config is generated with [DankMaterialShell](https://danklinux.com/).
There are a few manual changes to make it more fit to myself.  
The color pallet is gruvbox material (medium) from dms settings and I try to fit the 
rest to that.  

### Icons & colors

- For QT5 and QT6 apps I use the colors generated by dms and mutagen in the breeze stylethrough `qt6ct`.
- For GTK apps I use whatever mutagen generated in the adw-gtk3-dark.
- The Icons are [Buuf Nestort](https://git.disroot.org/eudaimon/buuf-nestort).


### "Default" applications I like to use with hyprland

- User shell: `fish`. (With [starship](https://starship.rs/))
- Terminal: `kitty`.
- Text editor: `neovim`. (With [NvChad](https://nvchad.com/))
- Media Player: `mpv`.
- File Manager: `PCManFM-qt`.
- App Launcher: `rofi`.
- Browser: `zen-browser` or `brave`.
- PDF reader: `okular` or `zathura`.

### Other

Other applications and programms I like to use:
- [`yt-dlp`](https://github.com/yt-dlp/yt-dlp) for downloading videos from youtube.
- [`ytcc`](https://github.com/woefe/ytcc) for checking on my favorite youtube channels.
- [`ttvchecker`](https://github.com/n00bady/ttvchecker) for checking on my favorit twitch streamers.
- [`lazygit`](https://github.com/jesseduffield/lazygit) TUI for git.
- [`fastfetch`](https://github.com/fastfetch-cli/fastfetch) very fast fetch program similar to neofetch.
- [`syncthing`](https://syncthing.net/) and [`syncthingtray-qt6`](https://github.com/Martchus/syncthingtray) for my file syncing needs.
- [`keepassxc`](https://keepassxc.org/) for all my passwords.
